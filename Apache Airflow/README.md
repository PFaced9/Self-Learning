# E-commerce Analytics Pipeline (Airflow Learning Project)

Learning project: orchestrating a daily e-commerce analytics pipeline with Apache Airflow 3.x.
Airflow coordinates; it does not do the heavy data processing itself.

Source: synthetic orders/customers/products/payments/shipments/returns data (Python/Faker).
Destinations: PostgreSQL (staging + operational tables) and ClickHouse (analytics tables).

See `ROADMAP.md` for the milestone plan.

Status: Milestone 0 in progress (environment setup).
